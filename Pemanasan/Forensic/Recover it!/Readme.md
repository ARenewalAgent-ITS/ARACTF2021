# Recover it!
---
## Deskripsi
Pada suatu hari PT. Telkom mendapat kiriman sebuah gambar dengan pesan "Gambar ini sudah dirusak dengan kunci kosong". Bantulah PT. Telkom untuk memecahkan misteri ini.
## Solusi
1. Peserta akan diberi sebuah gambar (sunset.jpg), seperti dibawah ini. Dalam gambar ini disisipi informasi untuk menuju ke gambar kedua <br/><img src="1.JPG" width="500px">
2. Mengekstrak informasi yang ada dalam gambar <br/><img src="2.JPG" width="500px">
3. Menuju dan mendownload gambar kedua yang ada dalam drive tersebut <br/><img src="3.JPG" width="500px">
4. Setelah di download gif tersebut belum bisa dibuka <br/><img src="4.JPG" width="500px">
5. Menggunakan hex editor untuk merecover file gif <br/><img src="5.JPG" width="500px">
6. Menyesuaikan hex dengan hex signature dari gif <br/><img src="6.JPG" width="500px"><br/><img src="7.JPG" width="500px">
7. Setelah itu maka gif dapat dibuka <br/><img src="8.JPG" width="500px">
8. Mendekripsi pesan dengan base 64 <br/><img src="9.JPG" width="500px">
